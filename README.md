Projet Arma Reforger Mapping by Arma III
  -Description:
    Réedition de la map Altis sur Arma reforger. 
      Le projet est de modifier la map Altis de Arma 3 pour intégrer Arma reforger. Il sera nécessaire de créer un environement pour convertir toute la map avec les nouveau items 3D de Arma reforger afin que la map soit identique a l'original.

  -Etapes:
    Création de l'IDE qui effectuera la conversion Arma III --> Arma reforger
